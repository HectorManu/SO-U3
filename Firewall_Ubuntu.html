Firewall


1. Entramos a la configuracion de red de Ubuntu




2. Aquí ingresamos a la preferencia de red y nos cambiamos como en la imagen siguiente para conectarnos a internet :)






3. Después de conectarnos a nuestra red, vamos a instalar el FireWalldesde la terminal de Ubuntu con el comando que se muestra en la imagen

apt-get install firewalld


Nota: debemos estar en superUsuario para ejecutar el comando.





5. Ahora instalamos el UFW que es un manejador del Firewall


Para instalar ejecutamos el comando apt-get install ufw





6. Ejecutamos el siguiente comando para pedir permiso para poder conectarnos




7. Activamos el Firewall con el comando que muestra la imagen




8. El comando siguiente nos da permiso de conectarnos al MYSQL




9. Con el comando ufw allow 22/tcp nos permite conectados al ssh




10. En este paso damos de alta los UDPS




11. El comando que se muestra en la imagen nos permite ver todos los comandos que tiene el servicio

ufw





12. Ejecutamos ufw status numbered si en caso de no funcionar, es por que el ufw esta inactivo

por lo que tenemos que activalor con el siguiente comando: ufw enable





13. con el ufw activado podemos ejecutar el ufw status numbered que nos permite visualizar que puertos

estan activados.


ALLOW IN nos indica que tiene permiso de entrada de cualquier sitio





14. El comando ufw firewalld status nos proporciona la informacion o el estado que tiene el servicio firewalld

como podemos observar nos dice que firewalld esta active(running)

este firewalld nos protege de los ataques (virus o hackers).
